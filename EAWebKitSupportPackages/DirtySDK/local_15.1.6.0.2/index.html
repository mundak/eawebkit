DirtySock SDK

 

The DirtySock SDK is a collection of client libraries, executables, tools, and sample applications that are geared around networked gaming.  The SDK includes the following basic components:

 

Contrib – Standalone networking libraries

DirtySock – Core networking libraries

Sample – Sample applications and utilities

 

For an overview of the DirtySock SDK and what it contains, please see the DirtySock SDK Overview PowerPoint presentation.
